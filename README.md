#### [The Commuting–Environment Nexus](https://github.com/salmanem25/Econ_396_Final_Project.git) | *Independent Research (R, Census/EPA/NOAA APIs)*
*An end-to-end data pipeline and statistical analysis testing associations between urban congestion, transit usage, and air quality.*

*   **Data Engineering:** Built a clean, 10-city, 7-year panel dataset from scratch by merging multi-source metrics across 3 public APIs; resolved API rate limits, JSON parsing, and real-world data gaps (e.g., matching NOAA weather station coordinates to city boundaries).
*   **Statistical Modeling:** Executed a pooled OLS regression in R, establishing a statistically significant positive correlation between vehicle congestion and PM2.5 levels, alongside a negative correlation with public transit share.
*   **Reporting:** Developed a fully automated, reproducible research compendium using R Markdown to generate dynamic visualizations and statistical narratives.
